# XuForge - Traveler's Armor Set

A Minecraft Mod for 1.21.1 (Fabric) that revives the Traveler's Gear from Tinkers' Construct 1 (MC 1.7.10).

## Features

This mod adds the Traveler's Armor Set, designed for explorers who need utility over raw defense.

    Traveler's Goggles
    Traveler's Vest
    Traveler's Wings
    Traveler's Boots
